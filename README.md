🔧 Example Use Case: Form Submission from a Website

🔄 Goal:

You have a contact form on your website. When someone submits it, you want to:



Capture the submitted data



Save it to Google Sheets



Send yourself a notification





You can do this using a Webhook Trigger.



🧪 Step-by-Step Example:

1\. Create Workflow in n8n

Drag in a Webhook Trigger node.



Set the HTTP Method to POST.



Give it a name like /contact-form.

Once saved, n8n will give you a Webhook URL, e.g.:

///

///////https://n8n.example.com/webhook/contact-form/////////



2\. Set Up Your Form to Send Data

Your website form might look like this in HTML:



html

<form action="https://n8n.example.com/webhook/contact-form" method="POST">

&nbsp; <input name="name" type="text" />

&nbsp; <input name="email" type="email" />

&nbsp; <textarea name="message"></textarea>

&nbsp; <button type="submit">Send</button>

</form>

When the form is submitted, a POST request is sent to n8n's webhook URL with the form data.
